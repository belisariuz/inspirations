# Inspiration Tab Mod
**Inspiration Tab** mod for RimWorld.

## What's this?
Ever wished you could just *give* your colonists an inspiration without waiting for the stars to align? Now you can! This mod adds a shiny new tab that lets you browse through all available inspirations and grant them to your colonists with a single click.

Whether you need a **Shoot Frenzy** to defend the colony or **Inspired Creativity** for that masterwork statue, we've got you covered.

## Features
- **Dedicated Tab**: A clean, easy-to-use interface.
- **Instant Granting**: Apply any inspiration instantly.
- **Mod Support**: Works with vanilla inspirations and should support modded ones too! 

## How to Install
1. Download the latest release.
2. Unzip into your RimWorld `Mods` folder.
3. Enable in the Mod Manager.
4. Enjoy!

## Compatibility
Should be compatible with everything.
